## 🔧 Customization Guide

### Change Colors
Edit CSS variables in `index.html` (line ~25):
```css
:root {
    --accent-gold: #ffd700;      /* Change primary color */
    --accent-blue: #00d4ff;      /* Change secondary color */
    --primary-dark: #0a0e27;     /* Change background */
}
```

### Update Company Info
Search for "Munene" and replace with your company name
Update email: support@munenearena.com
Update phone: +254 708 636 339

### Change Tournament Data
Find `app.tournaments` array in JavaScript section and update with your data

### Update Payment Methods
Search for "M-Pesa" in wallet section and update details

### Modify Favicon
Add this in `<head>`:
```html
<link rel="icon" type="image/svg+xml" href="data:image/svg+xml,...">
```

## 🐛 Troubleshooting

### Website Won't Load
- Check browser console (F12)
- Ensure Font Awesome CDN is accessible
- Try clearing browser cache

### Styling Issues
- Check CSS variable definitions
- Verify no browser extensions interfering
- Test in incognito/private mode

### Form Not Working
- Ensure JavaScript is enabled
- Check browser console for errors
- Verify localStorage is enabled

### Mobile Issues
- Test viewport is set correctly
- Clear device cache
- Test in different mobile browsers
